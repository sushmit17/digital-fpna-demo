# Digital FP&A Manager

**Agentic AI Demo** — Commercial Affiliate, Pharma

---

## What this demo shows

A working multi-agent AI system that automates two core FP&A workflows:

**Demo 1 — LBE Consolidation**
Upload 5 unit submission files → get a consolidated Affiliate P&L in seconds, with variances vs Prior LBE, Budget and Prior Year. The Validator Agent cross-checks every line.

**Demo 2 — S&OP Deck**
One command → CFO-quality variance narrative + 5-slide PowerPoint, ready to present.

---

## Agent architecture

```
You → Planner Agent → Executor Agent 1 or 2 → Validator Agent → Output
```
